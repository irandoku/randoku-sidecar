## Relationship to upstream

This project is not a replacement for Hermes Agent and is not an official Hermes Agent component. It began as a local sidecar experiment derived from `hermes-gpt`, then evolved toward a small, controlled AI operator sidecar focused on:

- explicit permission boundaries
- workspace-scoped capabilities
- auditable tool execution
- safe patch and test workflows
- human approval for high-risk operations

If upstream `hermes-gpt` later adopts similar ideas or becomes a broader collaborative project, compatible improvements may be contributed back or cherry-picked where appropriate.
